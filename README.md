1. Install Required Packages : pip install -r requirements.txt
2. Train the Machine Learning Models: python train\_models.py
3. Create the SQLite Database: python database.py
4. Run the Application: python app.py



After running, open the app in browser: http://127.0.0.1:5000/

Formatted table logs: http://127.0.0.1:5000/logs\_table
